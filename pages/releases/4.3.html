Web
Enhanced audio, rock on!
With great power comes great responsibility, when it comes to the Web. As there was audio issues when running games single-threaded, we updated the engine to support audio samples. This enabled us to leverage the Web Audio API in order to deliver seamlessly low-latency and high-quality audio.
